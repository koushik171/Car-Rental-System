# Car Images

Place car images in this folder with the following names:

1. toyota-camry.jpg
2. honda-civic.jpg
3. bmw-x3.jpg
4. mercedes-c-class.jpg
5. ford-focus.jpg
6. audi-a4.jpg
7. tesla-model3.jpg
8. hyundai-elantra.jpg

## Image Requirements:
- Format: JPG or PNG
- Recommended size: 400x250 pixels
- Aspect ratio: 16:10

## Where to get images:
- Free stock photos: unsplash.com, pexels.com
- Search for: "[car brand] [model] side view"
- Or use placeholder images (system will auto-generate if missing)

## Note:
The system will automatically show placeholder images if actual images are not found.
